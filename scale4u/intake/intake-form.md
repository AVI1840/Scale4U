# טופס קליטת יוזמה - Scale4U (ערוץ ללא רישיון)

מלא בקצרה ושלח למוביל Scale4U. אין צורך לדעת מה הפתרון - רק מה הכאב.

1. **מה הבעיה?** (בשפה שלך, 2-3 משפטים. מה כואב ביום-יום?)
2. **מי סובל מזה?** (תפקידים, כמה אנשים בערך, באיזו תדירות)
3. **איך זה מטופל היום?** (התהליך הקיים בקצרה)
4. **מה זה עולה לנו?** (זמן/כסף/שירות - גם הערכה גסה. אם לא יודע - כתוב "לא יודע")
5. **למה עכשיו?** (מה השתנה / מה ידחוף את זה)
6. **מי הבעלים?** (מי מוכן להשקיע יום השבוע כדי לקדם את זה)
7. **רגישות:** האם מעורב מידע אישי? מערכות ליבה? החלטות כלפי אזרח? (כן/לא/לא בטוח)

> מוביל Scale4U: הדבק טופס מלא לתוך Claude Code והרץ /s4u-new.
